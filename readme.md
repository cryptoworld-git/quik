# Quik Application Repo

A message service built around security, and flexibility to unify multiple platforms and message services into one easy to use system.  

Currently being re-written for new standards and services, see v2-Spec.txt for the breakdown before it's cleanly formatted into this readme.md



## Quik API Information

Quik is a 3-part system, relying on a Client, an Auth Hub, and a Message Relay in order to achieve full functionality.  
It is possible to only rely on some of these pieces, however with all 3 you gain an extremely enhanced experience.
